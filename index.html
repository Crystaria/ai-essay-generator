立即查看现有数据库

🧠 项目亮点

  • 支持自定义输入问卷，生成真实自然的英文个人陈述
  • 每次输出独一无二，控制查重率低于 5%
  • 自动集成查重、润色与 AI 检测模块
  • 通过 Notion + Zapier 实现无代码工作流

⚙️ 技术架构

  • 📄 Tally：收集用户输入
  • ⚡️ Zapier：自动触发文书生成 + 查重 + 写入 Notion
  • 📘 Notion：存储历史文书、比对相似度
  • 🤖 OpenAI GPT-4o：生成风格化英文文书
  • 🧪 伪 API 查重：基于语义差异分析输出相似度

📚 使用说明

  1. 打开 Tally 表单 并填写问卷
  2. 提交后系统自动生成文书并进行查重
  3. 结果将在 Notion 中保存，并提供相似度报告
  4. 如开启润色/AI检测模块，将自动推送处理
